# Challenge project - Build a minigame console app with GitHub Copilot

A simple console-based Rock, Paper, Scissors game implemented in `app.py`.

## Game details

- The player competes against the computer opponent.
- The computer chooses `rock`, `paper`, or `scissors` randomly each round.
- The player enters one of the valid choices in the terminal.
- Invalid entries are rejected and the player is prompted again.
- The player can also quit any round by entering `quit`, `q`, or `exit`.
- After each completed round, the player can choose whether to play again.
- When the game ends, the final score is displayed.

## How to run

From the repository root:

```bash
python3 app.py
```

## Controls

- Enter `rock`, `paper`, or `scissors` to play a move.
- Enter `quit`, `q`, or `exit` to end the game immediately.
- After each round, enter `y`/`yes` to play again or `n`/`no` to stop.
